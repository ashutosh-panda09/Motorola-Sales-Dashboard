# 📱 Mobile Sales Analysis Dashboard

## Description
The **Mobile Sales Analysis Dashboard** is a **Power BI** project. It helps analyze sales across brands, cities, months, payment methods, customer ratings and sales trends.

This dashboard turns raw sales data into to-understand visuals. It helps find business patterns and make decisions.

## Purpose
The goal of this dashboard is to show mobile sales performance. It helps find business insights like:
- overall sales performance
- top brands and models
- sales trends by month and day
- customer satisfaction patterns
- how customers pay
- sales by city

## Key Features

- **Sales Overview**

Shows total sales total items sold, total transactions and average price.

- **City-wise Sales Analysis**

Uses a map to show sales in cities.

- **Monthly Trend Analysis**

Highlights sales by month to find slow periods.

- **Brand Performance Comparison**

Compares top mobile brands by sales items sold and transactions.

- **Model-wise Sales Insights**

Finds the selling mobile models, by revenue.

- **Customer Rating Analysis**

Shows customer feedback and ratings.

- **Payment Method Analysis**

Displays how customers pay, like UPI, Debit Card, Cash and Credit Card.

- **Day- Sales Analysis**

Shows sales patterns by day of the week.

- **Interactive Filters**

Lets users filter and explore the data.

## Tools Used
- Microsoft Power BI
- DAX
- Power Query
- Data Visualization
- Data Modeling
